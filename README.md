# CorrLang experiment 


Welcome to the CorrLang experiment exercise!
This session is meant to give you a short introduction into _semantic interoperability_ (and to help
my master student Thea with finishing her thesis, who is working on "usability w.r.t. to semantic interoperability"
and is looking for some empiral data :wink:).

You should set aside approximately around _one hour_ to work on this assignment.
It is not a problem if you do not get throug all of the tasks since this is more like an experiment but please 
try to work on this approximately an hour in total.
Once, you are done, the most important part is fill out the survey that comes attached with Canvas exercise.
There, you can document your experience and also report technical issues.

The experiment comprises three exercises, each exercise comes with a markdown file that explains what you are supposed to do.
