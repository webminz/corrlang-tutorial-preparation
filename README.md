# CorrLang experiment 


Welcome to the CorrLang experiment exercise!
This session is meant to give you a short introduction into _semantic interoperability_ (and to help
my master student Thea with finishing her thesis, who is working on "usability w.r.t. to semantic interoperability"
and is looking for some empiral data :wink:).

You should set aside approximately around _one hour_ for this experiment.
Once, you are done, please fill out the attached form.

The task is to work through three exercises, each exercise comes with a similar markdown file that explains what you are supposed to do.
